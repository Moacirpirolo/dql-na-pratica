Queries prontas para copiar e usar

Você termina de ler e já aplica no mesmo dia. Assim é todo o conteúdo do guia.

dynatrace_notebook.dql
// Top 10 serviços com maior taxa de erro — últimas 24h
fetch logs, from: now()-24h
| filter severity == "ERROR"
| summarize
    total_errors = count(),
    by: {dt.entity.service}
| lookup [
    fetch dt.entity.service
    | fields id, entity.name
  ], sourceField: dt.entity.service, lookupField: id
| fields Servico = lookup.entity.name, Erros = total_errors
| sort Erros desc
| limit 10

// Latência p95 por endpoint com formatação em ms
fetch spans, from: now()-24h
| summarize
    p95 = percentile(duration, 95),
    by: {span.name}
| fields Endpoint = span.name,
    p95_ms = round(p95 / 1000000, decimals: 1)
| sort p95_ms desc
